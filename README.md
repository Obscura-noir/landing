# Sector 8 Landing

Лендинг для привлечения компаний в бета-версию платформы Sector 8.

## Стек
- Next.js 14+
- Tailwind CSS
- Framer Motion
- React Hook Form + Zod

## Запуск
```bash
npm install
npm run dev
```

## Структура
- `app/` — страницы и layout
- `components/` — UI-компоненты
- `lib/` — вспомогательные функции (интеграции)
- `styles/` — глобальные стили
